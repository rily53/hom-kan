# README

## users テーブル

| Column             | Type    | Options                   |
| ------------------ | ------- | ------------------------- |
| name               | string  | null: false               |
| email              | string  | null: false, unique: true |
| encrypted_password | string  | null: false               |

### Association

- has_many   :home_users
- has_many   :homes, through: :home_users
- has_many   :tasks
- has_many   :lists
- has_many   :memos
- has_many   :calendar_memos
- has_many   :informations
- has_many   :storehouses
